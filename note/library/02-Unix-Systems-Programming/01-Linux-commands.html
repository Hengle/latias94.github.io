Basic commands

  • file permissions

    • categories of access: user、group(may not include user)、others users
    • each file can be (r)eadable, (w)ritable, e(x)ecutable
    • expressed as octal numbers: 4 readable, 2 writable, 1 executable
      • 5 = readable + executable
      • 6 = readable + writable
      • 7 = all
      • 0 = nothing

  • cat – outputs its input

cat food2.txt
rice
pasta
veggies
rice
minties
minties
  • head – outputs only the first lines of its input
head -2 food2.txt
rice
pasta
  • tail – outputs only the last lines of its input
tail -1 food2.txt
minties
  • wc – prints newline, word, and byte counts for each file
wc food2.txt
6  6 40 food2.txt
  • echo - displays its string/argument and goes to a new line
    • -e 默认情况下，echo命令只显示可打印文本字符，要在echo命令中包含非可打印字符，比如制表符和换行符，必须用 -e 选项。
    • \c tells echo not to go to a new line
echo -e "Your username is: `whoami` \n"
Your username is: mzhuang
